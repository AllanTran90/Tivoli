## 🎯 Project Description
We will build a tivoli game where the user can place a bet and play a simple game (e.g. coin flip or dice). The game will be connected to the central bank API to handle money and stamps.

---

## 🎮 Game Idea
The user:
1. Chooses a stake (e.g. €5)
2. Plays the game (50/50 chance)
3. Wins or loses money
4. Receives a stamp after each play
