# Tivoli 🎡

A digital Tivoli amusement project built with Next.js, React and TypeScript.

Created by Allan Tran & Alex as part of the WU25 web development course.

---

# Games Included 🎮

## 🎯 Darts
- Real dartboard scoring
- 3 throws per round
- Wind system
- Throw history
- Reward payouts

## 🍫 Chocolate Wheel
- Spin wheel with betting system
- Random rewards
- Wallet integration
- Winning multipliers

## ⚡ Reaction Rush
- Timing-based reaction game
- Try to stop exactly at 10.00
- Reward system
- Score history

---

# Tech Stack 🛠️

- Next.js 16
- React
- TypeScript
- CSS Modules
- Canvas API

# Features ✨

- Mobile-first design

- Leaderboard system
- Responsive UI

Clone the repository:

```bash
git clone https://github.com/AllanTran90/Tivoli

install dependencies:

```bash
npm install
npm run dev
```
Open http://localhost:3000 in your browser to see the app.
